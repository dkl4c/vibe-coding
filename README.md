# 🌸 mako's Learning

> 漫步于樱花之森下的学习记录

一个简单而美丽的个人学习记录网站，使用纯原生 HTML + CSS + JavaScript 构建。

![日漫风格](https://img.shields.io/badge/风格-日漫风格-FFB7C5)
![原生开发](https://img.shields.io/badge/技术-原生开发-98D8C8)
![无框架](https://img.shields.io/badge/依赖-无框架依赖-98D8C8)

## ✨ 功能特色

### 📚 课程管理
- 展示个人课程列表
- 点击进入课程详情页
- 查看课程进度

### 📝 课程笔记
- 支持 Markdown 格式
- 按课程独立保存
- 编辑/预览切换
- 本地存储，不丢失

### 📋 作业展示
- 按课程分组
- 显示完成状态
- 链接到 GitHub 仓库

### 🎨 设计风格
- 樱花粉配色 (#FFB7C5)
- 薄荷绿点缀 (#98D8C8)
- 樱花飘落动画
- 柔和阴影与圆角

## 🚀 快速开始

### 查看网站
直接用浏览器打开 `index.html` 即可。

### 本地开发
推荐使用 VS Code 的 Live Server 扩展：
1. 安装 Live Server 扩展
2. 右键点击 `index.html`
3. 选择 "Open with Live Server"

## 📂 项目结构

```
├── index.html          # 首页
├── notes.html          # 课程笔记
├── homework.html       # 课程作业
├── help.html           # 帮助页面
├── styles.css          # 样式文件
├── script.js           # 脚本文件
└── modules/            # 模块详情页
    ├── module1.html
    ├── module2.html
    └── ...
```

## 💡 使用技巧

### 笔记
- 选择课程后开始记录
- 使用 Markdown 格式
- 点击保存按钮或关闭页面时自动保存

### 作业
- 点击"查看代码"跳转到 GitHub 仓库
- 使用标签切换不同课程的作业
